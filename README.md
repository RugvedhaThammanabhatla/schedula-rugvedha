Schedula - Doctor Appointment Booking System

Backend Internship Program Project built using NestJS, PostgreSQL, TypeORM, NeonDB and Render.

Project Setup

Clone the repository:

git clone https://github.com/RugvedhaThammanabhatla/schedula-rugvedha.git
cd schedula-rugvedha

Install dependencies:

npm install

Run the application:

npm run start:dev

Production:

npm run start

---

Environment Variables

Create a ".env" file and add:

DATABASE_URL=your_neon_database_url
JWT_SECRET=your_secret_key
PORT=3000
